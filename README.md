# MyGrowtopia Clone (C++ Engine)

This is a custom 2D Multiplayer Sandbox C++ Game Engine built using **Raylib** for rendering and **ENet** for networking.

## Features:
- Platformer Physics (Gravity, Jumping).
- AABB Tile Collision Constraints.
- UDP Multiplayer Synchronization.
- Block Breaking & Placing.
- Automated Android APK Cloud Compilation.
